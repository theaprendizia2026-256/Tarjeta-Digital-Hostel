# Contribuyendo a Tarjeta Digital - Atípico Hostel

Gracias por tu interés en mejorar este proyecto.

## Reglas de Oro

1. **Simplicidad ante todo:** Si una función no es crítica, no se agrega.
2. **Cero Dependencias:** No se aceptan frameworks pesados (React, Bootstrap, Tailwind). Todo se desarrolla en Vanilla (HTML, CSS y JS puro).
3. **Optimización Extrema:** Cada regla CSS y cada línea de JS debe justificarse por su eficiencia en recursos y rendimiento en dispositivos móviles.

Para proponer cambios, abre un **Pull Request** con una explicación técnica detallada de la mejora.
